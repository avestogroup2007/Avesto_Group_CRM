# Avesto Group CRM System — Фронтенд

Веб-интерфейс на React + Vite. Пока это интерактивный прототип (MVP) из ТЗ:
заявки с 5 фазами, роли Исполнитель/Контролёр, неизменяемый журнал, SLA-таймеры,
смены, SOP-чек-листы, контроль бюджетов, ИИ-маршрутизация, кассы, аналитика
продаж, дашборд директора и личная аналитика сотрудника. Двуязычный (RU / UZ).

## Стек

- **React 18** + **Vite 6**
- **Tailwind CSS 4** — стили
- **recharts** — графики
- **lucide-react** — иконки

## Запуск

```bash
cd frontend
npm install
npm run dev      # http://localhost:5173
```

Сборка продакшена:

```bash
npm run build    # результат в dist/
npm run preview  # локальный просмотр собранной версии
```

## Данные

Сейчас прототип хранит состояние в браузере: `store` работает через `window.storage`,
а в `src/main.jsx` подставлен shim на `localStorage`, чтобы данные сохранялись между
сессиями. На Этапе 8 плана бэкенда этот слой переключится на серверный API —
в `vite.config.js` уже настроен прокси `/api → http://localhost:3000`.

## Структура

```
frontend/
├── index.html          # входной HTML
├── vite.config.js      # Vite + React + Tailwind, прокси /api
├── src/
│   ├── main.jsx        # точка входа + localStorage-shim
│   ├── App.jsx         # приложение (прототип из ТЗ)
│   └── index.css       # подключение Tailwind
└── package.json
```

> Файл `src/App.jsx` — большой одностраничный прототип. По мере переноса на
> реальный бэкенд его имеет смысл разбить на модули и добавить линт/тесты.
